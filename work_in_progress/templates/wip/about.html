Это уже не просто прототип

Регаешься, тебя пускает внутрь с управлением.

Без регистрации только пластиковый куколдинг - сиди смотри, как другие красят.

В управлении настраиваешь группы статусов, настраиваешь статусы и в каком порядке они должны следовать друг за другом(опционально, если есть указание кто предыдущий, кто следующий кнопки для перевода в следующий статус выводятся). Один статус обязательно указать как "начальный", он будет использоваться как статус по-умолчанию при добавлении новых моделей черещ бота. Группы статусов нужны для "группировки" нескольких в пачки при отображении количества покрашенных/не покрашенных миниатюр

После добавления статусов можно создавать модели.

Жизненный цикл модельки определяется созданными статусами.

Добавляешь миньки в свой список. По мере работы надо минькой можно записывать затраченное время. Оно потом выводится в общей ленте и будет выводится в личном списке.

При записи времени можно указывать фотографию. Она будет отображаться в логе работы над моделью. Последняя фотография модели будет отображаться в общем списке моделей

Можно работать через веб морду, можно работать через бота в телеге. Чтобы работал бот, при регистрации надо указать юзернейм из телеграмма.
Бот: @fwip_bot
